# Stock Market Analysis with Yahoo Finance

## Project Overview

This project analyzes historical stock price data using Python and Yahoo Finance data. The goal is to compare major stocks based on price movement, total return, and volatility.

## Stocks Analyzed

- Apple (AAPL)
- Microsoft (MSFT)
- Nvidia (NVDA)
- Tesla (TSLA)
- Amazon (AMZN)

## Tools Used

- Python
- pandas
- yfinance
- matplotlib
- Git/GitHub
